cURL Examples

Test the webhook:

curl -X POST http://localhost:3000/api/webhook \ -H "Content-Type: application/json" \ -d '{"name":"John Doe","ic":"123456789","address":"123 Main St","property":"456 Property Ave","bank":"Test Bank","bankAddress":"789 Bank St","branchAddress":"321 Branch Rd","facility":"Home Loan"}'

Test the debug endpoint:

curl -X POST http://localhost:3000/api/test \ -H "Content-Type: application/json" \ -d '{"test":"data"}'

Test with invalid JSON:

curl -X POST http://localhost:3000/api/webhook \ -H "Content-Type: application/json" \ -d '{"invalid": json}'
